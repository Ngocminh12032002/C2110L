WritedPad Editor

WritedPad is an editor application that allows you to create Web pages easily and quickly.

Steps to open WritedPad application:

  1. Click Start
  2. Click Programs
  3. Click WritedPad

Advantages of using WritedPad editor:

  1. It allows faster creation of Web pages.
  2. It allows easier updation of text or images in WritedPad editor.
  3. It is platform independent.

Disadvantages of using a WritedPad editor:

  1. It consumes more memory space.
  2. It requires to fill-in more details such as meta data for each page.
